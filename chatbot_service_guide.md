## 📋 의뢰 전 준비사항

### 필요한 권한 (요약)

개발자에게 아래 권한을 부여해 주세요. **계정 공유 없이 초대 방식**으로 안전하게 진행됩니다.

| 플랫폼 | 필요 권한 | 용도 |
|--------|----------|------|
| 카카오톡 채널 관리자센터 | **매니저** | 채널 테스트, 메시지 확인 |
| 챗봇 관리자센터 | **정식 작업자** | 시나리오/블록/스킬 개발, 배포 |
| 카카오 디벨로퍼스 (앱) | **Editor** | API 연동 (Pro 플랜에만 필요) |

### 권한 부여 방법

1. **카카오톡 채널**: 관리자센터 → 관리 → '+관리자 초대' → 개발자 이메일 입력
2. **챗봇 관리자센터**: 관리자 → 관리자 초대 → 개발자 이메일 입력 → 정식 작업자 선택
3. **카카오 디벨로퍼스** (Pro 플랜): 앱 선택 → 팀 관리 → Editor 권한으로 초대

> 💡 개발 완료 후 권한을 회수하거나 Viewer로 변경할 수 있습니다.

## ❓ FAQ

<details>
<summary><b>Q1. 개발 기간은 얼마나 걸리나요?</b></summary>

플랜별 개발 기간은 다음과 같습니다.
- Basic: 2일
- Standard: 3일
- Pro: 5일

요구사항이 명확한 경우 기준이며, 상담/기획 기간은 별도입니다.

</details>

<details>
<summary><b>Q2. 개발 완료 후 수정이 가능한가요?</b></summary>

가능합니다. FAQ 추가/수정, 시나리오 변경 등은 별도 협의를 통해 진행합니다.

</details>

<details>
<summary><b>Q3. AI 챗봇의 정확도는 어떤가요?</b></summary>

Pro 플랜의 Gemini AI는 다음 기술을 사용합니다.
- 자연어 이해 및 생성
- 문맥 기반 응답
- Google Search Grounding (최신 정보 제공)

정확도는 지속적인 학습과 튜닝을 통해 개선됩니다.

</details>

<details>
<summary><b>Q4. 챗봇이 답변하지 못하는 경우는?</b></summary>

Fallback 메커니즘을 통해 처리합니다.
1. 상담원 연결 버튼 제공
2. 문의 접수 후 후속 연락
3. 미응답 질문을 분석하여 FAQ 업데이트

</details>

<details>
<summary><b>Q5. 카카오톡 채널이 없는데 가능한가요?</b></summary>

채널 개설이 선행되어야 합니다. 채널 개설부터 지원 가능하며, 소요 시간은 약 3~5일입니다.

</details>

<details>
<summary><b>Q6. 결제 연동도 가능한가요?</b></summary>

가능합니다 (Enterprise 플랜).
- 지원 PG: 토스페이먼츠, 나이스페이, KG이니시스 등
- 결제 방식: 카카오페이, 신용카드, 계좌이체

</details>

<details>
<summary><b>Q7. 챗봇 응답 속도는?</b></summary>

- Basic/Standard: 1초 이내
- Pro (AI): 2~4초

모든 플랜이 카카오 제약(5초)을 준수합니다.

</details>

<details>
<summary><b>Q8. 대화 내용 분석이 가능한가요?</b></summary>

**기본 분석** (모든 플랜)
- 일/주/월간 대화 건수
- 인기 질문 Top 10
- 시간대별 패턴

**고급 분석** (Pro)
- 사용자 의도 분류
- 미응답 질문 리스트
- 대화 완료율 분석

</details>

<details>
<summary><b>Q9. 다국어 지원이 가능한가요?</b></summary>

Pro 플랜에서 가능합니다.
- 지원 언어: 한국어, 영어, 일본어, 중국어 등
- 구현 방식: 언어 감지 + 다국어 응답

</details>
